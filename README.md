# Artisan Connect Market

## Project Description

Artisan Connect Market is a web application that connects local artisans with customers. Users can browse handmade products, explore artisan profiles, and support local businesses through an easy-to-use online marketplace.

## Features

- User Login
- Browse Products
- View Artisan Profiles
- Product Details
- Responsive Design

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Installation

1. Clone the repository

```bash
git clone https://github.com/Naseemashaik23/artisan-connect-market.git
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
